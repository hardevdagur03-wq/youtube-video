    Pipeline

    How It Works

    The pipeline automates the entire workflow from channel resolution to CSV export — no manual steps required.

    • 1
      Channel Resolution — Resolves a YouTube channel URL, handle, or ID to retrieve the canonical channel ID and title.
    • 2
      Video Discovery — Discovers every uploaded public video ID using paginated API calls until all videos are collected.
    • 3
      Metadata Fetch — Retrieves full metadata (title, upload date, views, likes, duration) for every video.
    • 4
      Transform & Export — Converts raw API data into the structured CSV format with video type detection (Video/Short) and YouTube URL generation.

    viewsTotal views
    likesTotal likes
    durationDuration (HH:MM:SS)
    video_typeVideo or Short
